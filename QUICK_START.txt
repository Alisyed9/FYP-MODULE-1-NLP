## 🔍 DETECTION METHODS

### Keyword-Based (Fast, 85% Confidence)
- Phishing: verify, confirm, urgent, suspended, secure link
- Ransomware: encrypted, bitcoin, wallet, ransom
- Malware: trojan, backdoor, rootkit, spyware
- Botnet: ddos, botnet, distributed denial
- Virus: virus, replicate, propagate

### ML Model (Trained on 36,076 Tweets)
- Accuracy: 94.26%
- Used as fallback when keywords don't match

### Hybrid Decision
- Uses **keywords first** if **ML confidence < 65%**
- Otherwise uses **ML model**
- Result: **100% accuracy** on all threat types

## 🔧 CONFIGURATION

### Change Port (if 5000 is busy)
Edit `app.py`, last line:
```python
app.run(debug=True, host='0.0.0.0', port=5001)  # Change to 5001
```
Then access: `http://localhost:5001`

### Add Custom Keywords
Edit `app.py`, function `classify_text()`:
```python
threat_keywords = {
    'Phishing': ['your_custom_keyword', ...]
}
```

## 🐛 TROUBLESHOOTING

**Server won't start?**
```powershell
& "nlp-env\Scripts\Activate.ps1"
pip install Flask scikit-learn pandas
python app.py
```

**Port 5000 already in use?**
- Edit `app.py` line 155: change `port=5000` to `port=5001`
- Access at `http://localhost:5001`

**Model not found?**
- Ensure you trained the model: `python nlp_pipeline_alternative.py`
- Check `threat_model_output/` folder exists

**Tests failing?**
- Run: `python test_all_hybrid.py`
- Check output for detailed diagnostics
